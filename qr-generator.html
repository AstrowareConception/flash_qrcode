Comment ça marche?

Cette page utilise la bibliothèque qrcode.js pour générer des QR codes à partir du texte que vous saisissez.

Voici les étapes principales:

  1. Saisissez le texte que vous souhaitez encoder dans le champ de texte
  2. Cliquez sur le bouton "Générer QR Code"
  3. Un QR code contenant votre texte sera généré et affiché
  4. Vous pouvez ensuite scanner ce QR code avec la page de scanner pour tester la fonctionnalité

Cette page complémentaire est utile pour tester le scanner QR code sans avoir besoin de trouver des QR codes ailleurs.
